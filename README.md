# Details 
* GTK Theme: Qogir Dark
* Icons: Tela Dark
* Font: Iosevka Nerd Font
* Terminal: Kitty
* Shell: ZSH
* Bar: Polybar
* Launcher: Rofi
* Editor: Neovim
* Wallpaper: https://cdna.artstation.com/p/assets/images/images/009/538/556/4k/inaki-andonegi-castle-in-the-sky.jpg
