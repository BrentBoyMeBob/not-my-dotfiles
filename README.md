# Installation

```
git clone --recursive https://github.com/smartsyncing/dotfiles
cd dotfiles
mv * ~/.config &&
mv ~/.config/.gtkrc-2.0 ~/
mv ~/.config/.zshrc ~/
openbox --reconfigure
source ~/.zshrc
```
